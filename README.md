## 📌 Descripción

Este proyecto es un **sistema hotelero** desarrollado en **C#**, diseñado para optimizar la administración de hoteles, hostales y alojamientos.

Permite gestionar:

- 🛏 Habitaciones  
- 📅 Reservas  
- 👥 Clientes  
- 💳 Pagos  
- 📊 Reportes  

---

## 🚀 Características

- Registro y control de habitaciones  
- Gestión de reservas (check-in / check-out)  
- Administración de clientes  
- Control de disponibilidad en tiempo real  
- Registro de pagos  
- Generación de reportes  
- Interfaz intuitiva  

---

## 🛠 Tecnologías utilizadas

- **Lenguaje:** C#  
- **Framework:** .NET / .NET Framework  
- **Base de datos:** SQL Server / MySQL  
- **Interfaz:** Windows Forms / WPF

## 💻 Instalación

### 1. Clonar el repositorio
```bash
git clone https://github.com/tu_usuario/sistema-hotelero.git
cd sistema-hotelero
2. Configurar la base de datos
Crear base de datos en SQL Server o MySQL
Ejecutar el script database.sql
3. Configurar conexión

Editar archivo de configuración:

App.config

Ejemplo:

<connectionStrings>
  <add name="conexion"
       connectionString="Server=localhost;Database=hotel;User Id=sa;Password=root;" />
</connectionStrings>
4. Ejecutar el proyecto
Abrir el proyecto en Visual Studio
Compilar la solución
Ejecutar la aplicación
